# 🔥 WinForge

**WinForge — أداة شاملة لتجهيز وتخصيص وتسريع جهاز Windows 10 / 11.**

متوفّرة بطريقتين:
- 🖱️ **تطبيق رسومي `WinForge.exe`** (موصى به): واجهة فيها **مربعات اختيار** تأشّر منها بالضبط أي لعبة/برنامج/خدمة/تويك تبي تعطّله أو تثبّته، مع أوضاع جاهزة وزر تطبيق وسجل مباشر.
- ⌨️ **سكربت `winforge.bat`**: نفس الوظائف عبر قائمة في نافذة الأوامر، لمن يفضّل CMD.

> 💡 التطبيق الرسومي يستهدف **.NET Framework 4.8** المثبّت مسبقاً على كل أجهزة Windows 10/11 — يشتغل بدون أي تثبيت.

## ✨ المميزات

- 🖥️🎮 **ثلاثة أوضاع: عمل / ألعاب / Basic** — البرنامج يسألك أول شي عن استخدام الجهاز ويتكيّف:
  - **عمل**: يحذف Xbox والألعاب، يعطّل Game DVR، أقصى تركيز.
  - **ألعاب**: يحتفظ بـ Xbox و Game Bar، يفعّل Game Mode + جدولة GPU + Ultimate Performance + تويكات الماوس والشبكة (winget يضيف Steam/Discord/OBS/Afterburner).
  - **Basic / Home**: لجهاز المنزل — حذف البرامج المعلّبة وإيقاف الإعلانات والتتبّع فقط، بدون تعطيل خدمات أو تويكات (الألطف والأأمن).
- 📊 **لوحة معلومات (Dashboard)** تفتح أول: نسخة وبناء Windows، المعالج وحِمله، الرام، كرت الشاشة، القرص، مدة التشغيل، وملخّص وش طبّق WinForge.
- 🔍 **ماسح حقيقي للبرامج** يعرض كل حِزم Appx المثبّتة فعلاً على جهازك **مع أحجامها** مرتّبة من الأكبر — تأشّر وتشيل.
- 🌐 **أدوات شبكة**: DNS سريع (Cloudflare/Google)، flush DNS، reset Winsock/TCP-IP.
- 🔄 **تحكّم بـ Windows Update** (إشعار فقط، منع إعادة التشغيل، استبعاد التعريفات، تأجيل الترقيات).
- 🪟 **تويكات Windows 11**: قائمة اليمين الكلاسيكية، محاذاة الشريط لليسار، إخفاء Widgets/Chat/البحث.
- 🔔 **فحص تحديثات WinForge تلقائياً** من GitHub مع زر تنزيل.
- 🧭 **قائمة تفاعلية (Menu)**: تختار وش تسوي بدل تشغيل كل شيء إجباري.
- 🎚️ **أوضاع جاهزة**: خفيف / متوسط / قوي — تتحكم بمقدار حذف البرامج.
- 🛡️ **نقطة استرجاع تلقائية** قبل أي تعديل — تقدر ترجع لو صار أي شيء.
- 🗑️ **حذف برامج مايكروسوفت الزائدة والألعاب**.
- 📦 **تثبيت برامج العمل تلقائياً (winget)**: Chrome، 7-Zip، Adobe Reader، VLC، Notepad++.
- ⚙️ **تعطيل الخدمات غير المهمة** في الخلفية → الجهاز يصير أخف.
- 🚫 **إيقاف الإعلانات والتتبّع** والاقتراحات في قائمة ابدأ.
- 🎮 **تعطيل Cortana و Game DVR**.
- 🧹 **تنظيف وأداء إضافي**: ملفات مؤقتة، خطة طاقة High Performance، مهام مجدولة، وإزالة OneDrive (اختياري).
- 🐧 **WSL / Startup**: تحديد ذاكرة WSL2/Docker عبر `.wslconfig` (يعالج استهلاك `VmmemWSL` العالي)، وتعطيل البرامج الثقيلة (Discord/Steam/Docker...) من بدء التشغيل.
- 📊 **عداد رام حي** داخل التطبيق + زر **Restart Explorer** لإعادة تشغيل واجهة ويندوز بسرعة.
- 💾 **حفظ/تحميل ملف تعريف مخصّص** (`.wforge`) لنقل اختياراتك بين الأجهزة.
- 📋 **قائمة بدء تشغيل ديناميكية** تعرض البرامج الفعلية على جهازك + زر **Docker Prune** لتنظيف Docker.
- 🔎 **بحث/فلترة** + **تحديد الكل/لا شيء** لكل تبويب + **عرض أحجام التطبيقات** + **ثيم ليلي/نهاري**.
- 📝 **سجل يُحفظ تلقائياً** + **شريط تقدّم فعلي (X من Y)** + **تأكيد بقائمة العناصر** قبل التطبيق + **SHA256** للتحقق.
- 🎛️ **وضعيات الجهاز بضغطة زر** (Gaming / Programming / Office): تعيد ضبط الجهاز كامل حسب استخدامك اليوم، وقابلة للعكس بمجرد تفعيل وضع آخر.
- 📝 **ملف سجل (Log)** يوثّق كل عملية بجانب السكربت.
- ↩️ **سكربت تراجع** `winforge-undo.bat` يعيد الإعدادات لوضعها الافتراضي.
- 🔐 **يطلب صلاحيات المسؤول تلقائياً**.

## 🚀 طريقة الاستخدام

### الطريقة الرسومية (موصى بها)
1. نزّل **`WinForge.exe`** من صفحة [**Releases**](../../releases).
2. **كليك يمين** ← **Run as administrator**.
3. اختر وضع جاهز (Office / Gaming / Basic) أو أشّر يدوياً على اللي تبيه من التبويبات.
4. تأكد أن **Create restore point** مفعّل، ثم اضغط **APPLY SELECTED**.
5. لو احتجت ترجع: اضغط **Revert settings (Undo)**.

### الطريقة عبر CMD
1. حمّل ملف **`winforge.bat`** على الجهاز.
   - أو نزّله جاهزاً من صفحة [**Releases**](../../releases).
2. اضغط عليه **كليك يمين** ← **Run as administrator** (تشغيل كمسؤول).
3. تطلع لك **قائمة** — اختر منها:
   - `[1]` تنظيف سريع (موصى به) — وضع متوسط.
   - `[2]` تنظيف مع اختيار الوضع (خفيف / متوسط / قوي).
   - `[3]` تثبيت برامج العمل (winget).
   - `[4]` تنظيف وأداء إضافي.
   - `[5]` إعداد كامل (تنظيف + تثبيت + تنظيف إضافي).
4. انتظر حتى ينتهي، ثم اختر **إعادة التشغيل** لتطبيق كل التغييرات.

> 📝 يُحفظ ملف سجل `winforge_*.log` بجانب السكربت يوثّق كل ما تم.

## 🔁 الرجوع للوضع السابق

- **تراجُع عن الإعدادات والخدمات:** شغّل **`winforge-undo.bat`** (كليك يمين ← Run as administrator) — يعيد الخدمات والإعدادات لوضعها الافتراضي.
- **استرجاع كامل:** افتح *Create a restore point* ← *System Restore* واختر النقطة باسم **`Before WinForge`**.
- **إعادة برنامج محذوف:** ثبّته من **Microsoft Store**.
