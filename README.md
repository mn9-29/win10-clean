# 🧹 win10-clean

**سكربت CMD لتنظيف وتجهيز جهاز Windows 10 للعمل — خفيف، سلس، وبدون مشاكل.**

تشغّله مرة واحدة فقط، ويقوم بكل شيء تلقائياً: يحذف البرامج والألعاب الزائدة من مايكروسوفت، يعطّل التتبّع والإعلانات، ويحافظ على المتصفح والأوفس وأدوات العمل.

---

## ✨ المميزات

- 🛡️ **نقطة استرجاع تلقائية** قبل أي تعديل — تقدر ترجع لو صار أي شيء.
- 🗑️ **حذف برامج مايكروسوفت الزائدة والألعاب**.
- ⚙️ **تعطيل الخدمات غير المهمة** في الخلفية → الجهاز يصير أخف.
- 🚫 **إيقاف الإعلانات والتتبّع** والاقتراحات في قائمة ابدأ.
- 🎮 **تعطيل Cortana و Game DVR**.
- ⚡ **تحسينات أداء خفيفة**.
- 🔐 **يطلب صلاحيات المسؤول تلقائياً**، وآمن للتشغيل مرة واحدة.

---

## ✅ البرامج التي يحافظ عليها

| البرنامج | السبب |
|---|---|
| Microsoft Edge | المتصفح |
| Microsoft Store | المتجر (لإعادة تثبيت أي برنامج) |
| Microsoft Office | كل برامج الأوفس |
| الآلة الحاسبة، الصور، Notepad، الرسام، أداة القص، Sticky Notes | أدوات العمل الأساسية |

## 🗑️ البرامج التي يحذفها

Xbox وكل توابعه • Candy Crush والألعاب • Bing (أخبار/طقس/رياضة) • 3D Builder و 3D Viewer • Skype • Maps • People • Groove Music و Movies&TV • Mixed Reality • Get Help و Tips • وكل التطبيقات المعلّبة مثل Netflix / Spotify / TikTok / Facebook ... إلخ.

---

## 🚀 طريقة الاستخدام

1. حمّل ملف **`win10-clean.bat`** على الجهاز.
   - أو نزّله جاهزاً من صفحة [**Releases**](../../releases).
2. اضغط عليه **كليك يمين** ← **Run as administrator** (تشغيل كمسؤول).
3. اقرأ الرسالة ثم اضغط أي زر للمتابعة (أو `Ctrl + C` للإلغاء).
4. انتظر حتى ينتهي، ثم اختر **إعادة التشغيل** لتطبيق كل التغييرات.

---

## 🔁 الرجوع للوضع السابق

- **استرجاع كامل:** افتح *Create a restore point* ← *System Restore* واختر النقطة باسم **`Before win10-clean`**.
- **إعادة برنامج محذوف:** ثبّته من **Microsoft Store**.

---

## 📦 الإصدارات والتحديثات

- كل تحديث (إصلاح أو ميزة جديدة) يُنشر في صفحة [**Releases**](../../releases) برقم إصدار واضح.
- نتبع نظام [Semantic Versioning](https://semver.org/lang/ar/): **MAJOR.MINOR.PATCH**
  - **PATCH** (1.0.0 → 1.0.1): إصلاح خطأ.
  - **MINOR** (1.0.0 → 1.1.0): إضافة ميزة جديدة.
  - **MAJOR** (1.0.0 → 2.0.0): تغيير كبير.
- راجع ملف [`CHANGELOG.md`](./CHANGELOG.md) لتفاصيل كل إصدار.

## ⚠️ ملاحظات مهمة

- مُصمَّم لـ **Windows 10** (يعمل أغلبه على Windows 11 مع اختلاف بعض الأسماء).
- إذا كان جهاز الشركة مُدار بسياسات (**Domain / Intune**)، بعض الإعدادات قد يُعيدها مسؤول النظام — يُفضّل التنسيق مع قسم تقنية المعلومات.
- آمن للتشغيل مرة واحدة (Preset) ويمكن إعادة تشغيله بدون مشاكل.
