# win10-clean

سكربت CMD لتنظيف وتجهيز جهاز **Windows 10** للعمل بشكل خفيف وسلس.

## ماذا يفعل السكربت؟

| العملية | التفاصيل |
|---------|----------|
| 🛡️ نقطة استرجاع | يُنشئ **System Restore Point** قبل أي تعديل (للأمان — تقدر ترجع لو صار شي) |
| 🗑️ حذف البرامج الزائدة | يحذف الألعاب وبرامج مايكروسوفت اللي ما لها فائدة (Xbox، Candy Crush، Bing، 3D، Skype، Maps، People... إلخ) |
| ⚙️ تعطيل الخدمات | يوقف خدمات الخلفية غير المهمة (Xbox، التتبّع/Telemetry، الخرائط...) |
| 🚫 تعطيل الإعلانات والتتبّع | يوقف الإعلانات، الاقتراحات في قائمة ابدأ، معرّف الإعلانات، وسجل النشاط |
| 🎮 تعطيل Cortana و Game DVR | يوقف كورتانا وتسجيل الألعاب بالخلفية لتوفير موارد الجهاز |
| ⚡ تحسينات الأداء | تقليل تأخير بدء التشغيل، إظهار امتدادات الملفات، إيقاف إعلانات شاشة القفل |

## ✅ البرامج اللي يحافظ عليها (ما يحذفها)

- **Microsoft Edge** (المتصفح)
- **Microsoft Store** (المتجر)
- **Microsoft Office** (كل برامج الأوفس)
- الآلة الحاسبة، الصور، Notepad، الرسام، أداة القص، Sticky Notes

## 🚀 طريقة الاستخدام

1. حمّل الملف `win10-clean.bat` على الجهاز.
2. اضغط عليه **كليك يمين** ← **Run as administrator** (تشغيل كمسؤول).
3. اقرأ الرسالة واضغط أي زر للمتابعة (أو `Ctrl + C` للإلغاء).
4. انتظر حتى ينتهي، ثم اختر **إعادة التشغيل** لتطبيق كل التغييرات.

> 💡 السكربت يطلب صلاحيات المسؤول تلقائياً، فلو شغّلته عادي بيطلع لك نافذة UAC للموافقة.

## 🔁 الرجوع للوضع السابق

لو احتجت ترجع أي شيء:
- افتح **Create a restore point** ← **System Restore** واختر النقطة باسم `Before win10-clean`.
- البرامج المحذوفة تقدر تعيد تثبيتها من **Microsoft Store**.

## ⚠️ ملاحظات

- مُصمَّم لـ **Windows 10**. على Windows 11 أغلب الأشياء تشتغل لكن قد تختلف بعض الأسماء.
- إذا جهاز الشركة مُدار بسياسات (Domain / Intune)، بعض الإعدادات قد تُعاد من قِبل المسؤول.
- آمن للتشغيل مرة واحدة (Preset)، ويمكن إعادة تشغيله بدون مشاكل.
