# سجل التغييرات (Changelog)

كل التغييرات المهمة في هذا المشروع تُوثَّق هنا.
يتبع المشروع [Semantic Versioning](https://semver.org/lang/ar/):
**MAJOR.MINOR.PATCH** — إصلاح = PATCH، ميزة جديدة = MINOR، تغيير كبير = MAJOR.

## [1.2.0] - 2026-06-20
### مميزات جديدة
- **وضعان عند بدء التشغيل: عمل / ألعاب** — البرنامج يسألك عن استخدام الجهاز أولاً.
- **وضع الألعاب (Gaming)**:
  - يحتفظ بـ Xbox و Game Bar وخدماتها (لا يحذفها).
  - يفعّل Game Mode + جدولة GPU (HAGS) + خطة طاقة Ultimate Performance.
  - يعطّل تسريع الماوس وتأثيرات الواجهة (FPS أعلى).
  - يقلّل زمن الاستجابة بتعطيل Nagle على كل واجهات الشبكة.
  - تثبيت winget يضيف Steam و Discord تلقائياً.
- **وضع العمل** يبقى كما هو (حذف Xbox، تعطيل Game DVR...).
- سكربت التراجع `win10-clean-undo.bat` صار يعكس تويكات الألعاب أيضاً.

## [1.1.0] - 2026-06-20
### مميزات جديدة
- **قائمة تفاعلية (Menu)**: تختار وش تسوي بدل تشغيل كل شيء إجباري.
- **أوضاع جاهزة (Profiles)**: خفيف / متوسط / قوي — تتحكم بمقدار حذف البرامج.
- **تثبيت برامج العمل عبر winget**: Chrome، 7-Zip، Adobe Reader، VLC، Notepad++ (القائمة قابلة للتعديل).
- **تنظيف وأداء إضافي**: حذف الملفات المؤقتة وسلة المحذوفات، خطة طاقة High Performance، تعطيل مهام التتبّع المجدولة، وإزالة OneDrive (اختياري).
- **ملف سجل (Log)**: يوثّق كل عملية في ملف `.log` بجانب السكربت.
- **سكربت تراجع** `win10-clean-undo.bat`: يعيد الخدمات والإعدادات إلى وضعها الافتراضي.

## [1.0.1] - 2026-06-20
### إصلاحات
- **إصلاح خطأ يوقف السكربت في الخطوة الأولى** (إنشاء نقطة الاسترجاع): كان سبب الخطأ تقسيم أوامر PowerShell على عدة أسطر بعلامة `^`. الآن كل أوامر PowerShell في سطر واحد.
- نقطة الاسترجاع لم تعد توقف السكربت إذا فشلت — يظهر تنبيه ويكمل (`try/catch`).
- إزالة قيد الـ 24 ساعة (`SystemRestorePointCreationFrequency`) حتى تُنشأ نقطة الاسترجاع دائماً.
- حذف البرامج صار عبر استدعاء PowerShell واحد لكل القائمة (أسرع وأكثر استقراراً).
- رسالة واضحة لو تعذّر رفع الصلاحيات تلقائياً، مع إبقاء النافذة مفتوحة.

## [1.0.0] - 2026-06-20
### الإصدار الأول
- إنشاء نقطة استرجاع (System Restore Point) قبل أي تعديل.
- حذف برامج مايكروسوفت الزائدة والألعاب (Xbox, Bing, Candy Crush, 3D, Skype, Maps, People...).
- تعطيل خدمات الخلفية غير المهمة (Xbox, DiagTrack/Telemetry, Maps...).
- تعطيل الإعلانات والتتبّع والاقتراحات في قائمة ابدأ.
- تعطيل Cortana و Game DVR.
- تحسينات أداء خفيفة (تقليل تأخير الإقلاع، إظهار امتدادات الملفات، إيقاف إعلانات شاشة القفل).
- الحفاظ على Edge و Microsoft Store و Office والأدوات الأساسية.
