# سجل التغييرات (Changelog)

كل التغييرات المهمة في هذا المشروع تُوثَّق هنا.
يتبع المشروع [Semantic Versioning](https://semver.org/lang/ar/):
**MAJOR.MINOR.PATCH** — إصلاح = PATCH، ميزة جديدة = MINOR، تغيير كبير = MAJOR.

## [2.9.0] - 2026-06-21
### صيانة + نسخ احتياطي + واجهة عربية 🧰🛡️🌍
- **تبويب Maintenance جديد**: فحص ملفات النظام (SFC)، إصلاح صورة Windows (DISM)، فحص القرص (chkdsk)، تنظيف القرص، تحسين/TRIM، مسح كاش Windows Update، وإعادة ضبط الشبكة — كلها بضغطة Apply.
- **نسخ احتياطي للريجستري**: زر "Backup registry now" يصدّر الفروع اللي يعدّلها WinForge إلى مجلد مؤرّخ في Documents، وزر "Restore .reg" يسترجعها، مع خيار **"انسخ الريجستري أيضاً" قبل التطبيق** للتراجع الدقيق.
- **واجهة عربية كاملة بضغطة زر** 🌍: زر "العربية/English" يبدّل كل نصوص الواجهة (الأزرار، التبويبات، التلميحات) ويقلب الاتجاه إلى RTL.

## [2.8.0] - 2026-06-21
### إضافات كبيرة للتطبيق الرسومي 🚀
- **تبويب Dashboard جديد** (يفتح أول): يعرض حيّاً نسخة وبناء Windows، اسم الجهاز، المعالج وحِمله، الرام، كرت الشاشة، مساحة القرص، ومدة التشغيل — مع **ملخّص وش طبّق WinForge** (تلميتري، Cortana، Game DVR، الإعلانات...).
- **ماسح حقيقي للبرامج** (تبويب "Installed (scan)"): يمسح جهازك الفعلي ويعرض **كل حِزم Appx القابلة للإزالة مع أحجامها** مرتّبة من الأكبر — تأشّر اللي تبي تشيله وتطبّق.
- **تبويب Network**: DNS سريع (Cloudflare / Google / تلقائي)، flush DNS، release/renew، reset Winsock و TCP/IP.
- **تبويب Updates**: إيقاف التحديث التلقائي (إشعار فقط)، منع إعادة التشغيل وأنت مسجّل دخول، استبعاد تحديثات التعريفات، تأجيل ترقيات الميزات.
- **تبويب Win 11 / UI**: قائمة اليمين الكلاسيكية، محاذاة الشريط لليسار، إخفاء Widgets/Chat/مربع البحث، إظهار الثواني بالساعة.
- **حجب التلميتري عبر ملف hosts** (في تبويب Privacy، اختياري).
- **فحص تحديثات WinForge تلقائياً** من GitHub مع شريط تنبيه وزر تنزيل + زر "Check updates".
- زر Undo صار يرجّع أيضاً تويكات Windows Update وقائمة Win11 الكلاسيكية.

## [2.7.0] - 2026-06-21
### إعادة تسمية 🔥
- **تغيير اسم المشروع من `win10-clean` إلى `WinForge`** ليعكس إنه أداة شاملة لتجهيز وتخصيص الجهاز (وليس مجرد تنظيف، ويدعم Windows 10 و 11).
- التطبيق صار `WinForge.exe`، والسكربتات `winforge.bat` و `winforge-undo.bat`.
- تحديث الـ namespace والواجهة والتوثيق و workflow الإصدارات للاسم الجديد.

## [2.6.0] - 2026-06-20
### وضعيات الجهاز بضغطة زر 🎛️
- **تبويب جديد "Device Modes"** فيه 3 أزرار كبيرة تعيد ضبط الجهاز كامل حسب استخدامك (قابلة للعكس — تبدّل بينها وقت ما تشاء):
  - **Gaming Mode**: خطة طاقة High Performance + تفعيل Game Mode + إيقاف Docker/WSL وأدوات التطوير لتفريغ الرام للألعاب.
  - **Programming Mode**: High Performance + إغلاق برامج الألعاب (Steam/Discord/Epic) + تشغيل Docker Desktop و WSL.
  - **Office Mode**: خطة Balanced + إغلاق برامج الألعاب و Docker/WSL لجهاز خفيف وهادئ.
- لا تحذف شيئاً — فقط تشغّل/توقف البرامج والخدمات وتضبط الطاقة، مع تأكيد وسجل وشريط تقدّم.

## [2.5.0] - 2026-06-20
### مميزات جديدة
- **حفظ السجل تلقائياً** في ملف `win10-clean_*.log` بجانب التطبيق (كل العمليات موثّقة بالوقت).
- **شريط تقدّم فعلي (X من Y)**: يعرض التقدّم الحقيقي واسم العنصر الجاري تنفيذه أثناء التطبيق.
- **تأكيد بقائمة العناصر**: قبل التطبيق يعرض قائمة بكل اللي راح ينفّذ بالضبط للمراجعة.
- **أيقونة محسّنة** (تصميم أنظف + أحجام حتى 256px) و**ملف SHA256** مرفق في الـ Releases للتحقق من سلامة الـ `.exe`.

## [2.4.0] - 2026-06-20

## [2.3.0] - 2026-06-20
### مميزات جديدة
- **أيقونة احترافية للتطبيق** (`app.ico`) تظهر في شريط المهام والنافذة.
- **حفظ/تحميل ملف تعريف مخصّص** (`Save profile` / `Load profile`): تحفظ اختياراتك في ملف `.w10c` وتحمّلها على أي جهاز.
- **قائمة بدء تشغيل ديناميكية**: تبويب WSL/Startup صار يعرض **برامج بدء التشغيل الفعلية على جهازك** (يقرأها من الريجستري) بدل قائمة ثابتة، وتأشّر منها أي واحد تبي تعطّله.
- **زر "Docker Prune"**: ينظّف Docker (`docker system prune -f`) من الحاويات والصور غير المستخدمة بضغطة، مع عرض المساحة قبلها.

## [2.2.0] - 2026-06-20
### مميزات جديدة
- **عداد رام حي** في رأس التطبيق (يُحدّث كل ثانيتين، ويتلوّن أخضر/برتقالي/أحمر حسب الضغط).
- **زر "Restart Explorer"** لإعادة تشغيل واجهة ويندوز بدون ريستارت كامل (يحرّر ذاكرة Explorer).
- **خطوة توقيع تلقائية (اختيارية) في CI**: تُوقّع الـ `.exe` إذا أضفت السرّين `CERT_PFX_BASE64` و `CERT_PASSWORD`؛ وإلا تُتخطّى.
  - ملاحظة: تحذير **SmartScreen** لا يختفي إلا بشهادة **OV/EV** من جهة موثوقة (وبعد بناء سمعة) — الشهادة ذاتية التوقيع لا تنفع.

## [2.1.0] - 2026-06-20
### مميزات جديدة (WSL / Startup)
- **تبويب جديد "WSL / Startup"** في التطبيق الرسومي (وخيار `[7]` في السكربت):
  - **Optimize WSL2 memory**: ينشئ ملف `.wslconfig` يحدّد ذاكرة WSL/Docker بـ 8GB ويفعّل `autoMemoryReclaim` (يرجّع الذاكرة المحرّرة لويندوز تلقائياً)، ثم يطفي WSL.
  - **Shut down WSL now**: يوقف WSL2 فوراً لتحرير الرام.
  - **تعطيل البرامج الثقيلة من بدء التشغيل**: Docker، Discord، Steam، SteelSeries، GitHub Desktop، Epic، Spotify (قابل للتراجع من Task Manager).
- تعالج استهلاك `VmmemWSL` العالي وتكدّس برامج بدء التشغيل في الذاكرة.

## [2.0.0] - 2026-06-20
### تطبيق رسومي جديد (GUI) 🎉
- **برنامج `win10-clean-gui.exe` بواجهة رسومية** (C#/.NET WPF) — للتحكّم بالضغط بدل الأوامر:

  - **تبويبات منظّمة**: Apps، Privacy، Services، Gaming، Performance، Cleanup، Install.
  - **خيار نقطة الاسترجاع** + **سجل مباشر** للعمليات + زر **Revert (تراجع)**.
- يستهدف **.NET Framework 4.8** المثبّت مسبقاً على كل أجهزة Windows 10/11 (يشتغل بدون تثبيت).
- **بناء تلقائي عبر GitHub Actions**: يُجمّع الـ `.exe` ويُرفق في صفحة Releases مع كل إصدار.
- ملفات `win10-clean.bat` و `win10-clean-undo.bat` تبقى متاحة لمن يفضّل سطر الأوامر.

## [1.0.1] - 2026-06-20
### إصلاحات
- **إصلاح خطأ يوقف السكربت في الخطوة الأولى** (إنشاء نقطة الاسترجاع): كان سبب الخطأ تقسيم أوامر PowerShell على عدة أسطر بعلامة `^`. الآن كل أوامر PowerShell في سطر واحد.
- نقطة الاسترجاع لم تعد توقف السكربت إذا فشلت — يظهر تنبيه ويكمل (`try/catch`).
- إزالة قيد الـ 24 ساعة (`SystemRestorePointCreationFrequency`) حتى تُنشأ نقطة الاسترجاع دائماً.
- حذف البرامج صار عبر استدعاء PowerShell واحد لكل القائمة (أسرع وأكثر استقراراً).
- رسالة واضحة لو تعذّر رفع الصلاحيات تلقائياً، مع إبقاء النافذة مفتوحة.
